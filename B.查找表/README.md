# 查找表问题
1.查找有无
元素 'a' 是否存在 set 集合
2.查找对应关系（键值对应）
元素 'a' 出现了几次  map 字典

注意map  set 底层实现   ：
哈希：不能保证数据顺序性
二分搜索树 （数据的顺序性） 
       数据集中的的最大值和最小值
       某个元素的前驱和后继
       某个元素的floor和ceil
       某个元素的rank
       选择某个排位元素的select
        

                普通数组    顺序数组        二分搜索树（平衡）   哈希表
    插入          O(1)        O(N)            O(LogN)         O(1)
    
    查找          O(N)        O(LogN)         O(LogN)         O(1)
    
    删除          O(N)        O(N)            O(LogN)         O(1)
