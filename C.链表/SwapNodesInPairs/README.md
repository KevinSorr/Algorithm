# 链表问题

LeetCode 24 Swap Nodes in Pairs
给定一个链表  对于两个相邻的节点 交换其位置
如：链表为1--->2--->3--->4--->null
  返回：2 --->1--->4--->3-->null
  只能对节点修改  不能修改节点的值
  
LeetCode 25 Reverse Nodes in K-Group
给定一个链表，每k个节点为一组，反转每一组的k个节点。k为正整数且小于等于链表长度
如果链表长度不是k的整数倍 剩余部分不需要进行反转 
如： 1--->2--->3--->4--->5
k=2 返回：2--->1--->4--->3--->5
k=3 返回：3--->2--->1--->4--->5
